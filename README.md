# БК-СТРОЙ - Flask приложение

Flask приложение для строительной компании БК-СТРОЙ. Дизайн конвертирован из React в чистый HTML/CSS.

## Установка

1. Установите зависимости:
```bash
pip install -r requirements.txt
```

2. Запустите приложение:
```bash
python app.py
```

3. Откройте браузер и перейдите по адресу:
```
http://localhost:5000
```

## Структура проекта

```
BkStroy/

├── requirements.txt       # Зависимости Python
```
